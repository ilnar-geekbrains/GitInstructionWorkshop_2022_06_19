# Что такое ветки и для чего они нужны при работе с системой контроля версий

Ветка — это копия проекта, созданная для внесения отдельностоящих изменений. Также, обязательно существует основная защищенная ветка (main), в которую сливают изменения из других веток.
Чаще всего, под определенную задачу выделяют отдельную ветку (в свою очередь, она может разделяться на другие ветки для подзадач), а после выполнения задачи и тестирования проекта с этими изменениям, выполняют слияние этой ветки с main-веткой.
